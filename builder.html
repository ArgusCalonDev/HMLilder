Output

  • Setelah selesai mengisi kolom kolom yang ada, copy paste script atau download file html.
  • Jika menggunakan metode copy paste, buka editor teks, seperti notepad, lalu lanjut ke langkah 4.
  • Jika menggunakan metode download, ubah nama file menjadi index.html untuk file beranda. selain beranda, nama file bebas.
  • Mulai modifikasi file. Petunjuk tiap bagian: disini
  • Jika sudah, selamat website sudah jadi! Upload ke hosting, seperti Hostinger, Domainesia, Cloudflare, atau Netlify agar website anda bisa online. Anda bisa mengikuti tutorial di internet karena cara mengupload file ke tiap hosting berbeda beda.


  • <!DOCTYPE html> = deklarasi file HTML.
  • <html lang="id"> = bahasa file HTML. "id" = Indonesia.
  • <head> = bagian kepala pada website, tidak terlihat di badan halaman.
  • <meta charset="UTF-8"> = tag meta untuk encoding file (UTF-8), agar karakter unik seperti emoji bisa tampil.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> = tag meta untuk halaman responsif di layar kecil seperti HP.
  • <title>Nama Bisnis Anda - Tagline</title> = judul halaman di bagian atas browser.
  • <style>...</style> = elemen CSS untuk atur file agar rapi dan enak dipandang.
  • </head> = penutup tag head
  • <body> = bagian badan halaman, yang terlihat oleh pengguna.
  • <div class="container"> = "div" menandakan suatu bagian tertentu di halaman. class="..." memberi tanda pada suatu elemen, agar bisa diatur lewat CSS dan javascript.
  • <header style="text-align: center; padding: 20px 0;"> = header, bagian paling atas website.
  • Didalam header, berisi tag untuk gambar (<img src="letak-gambar.ekstensi" alt="nama bisnis anda">) jika Anda mengunggah gambar ke dalam halaman. Jika tidak, tag ini kosong.
  • </header> = penutup tag header
  • <main> = konten utama
  • isi dari tag main adalah konten utama yang terlihat di preview page builder.
  • </main> = penutup main.
  • <footer> = bagian paling bawah website
  • ... isi dari footer
  • </footer> = penutup footer
  • </div> = penutup tag bagian "container" dalam halaman.
  • <script>...</script> = berisi kode javascript.
  • </body> = penutup badan halaman.
  • </html> = penutup halaman HTML.
