Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup dasar yang digunakan untuk membuat dan menyusun struktur halaman website. HTML berfungsi untuk menentukan elemen-elemen utama dalam sebuah halaman, seperti judul, paragraf, gambar, tautan, tabel, dan formulir, agar dapat ditampilkan dengan benar di browser. Dengan HTML, pengembang dapat mengatur struktur dan urutan konten pada halaman website, namun HTML tidak digunakan untuk mengatur tampilan visual atau interaksi lanjutan.

Oleh karena itu, HTML biasanya digunakan bersama CSS untuk mengatur tampilan, serta JavaScript untuk menambahkan interaksi pada website.

Bagaimana Cara Membuat Website HTML?

Berikut adalah langkah-langkah membuat website HTML dasar untuk pemula.

  1. Pilih Editor Teks
    Gunakan editor teks seperti Notepad (Windows), TextEdit (Mac), atau editor kode seperti Visual Studio Code, Sublime Text, dan Atom.
  2. Buat File HTML Baru
    Buat file baru dan simpan dengan ekstensi .html. Jika file ini adalah halaman utama (beranda), beri nama index.html.
  3. Tulis Struktur Dasar HTML 
    <!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
</body>
</html>
  4. Penjelasan Tiap Bagian
    <!DOCTYPE html> → Menandakan bahwa dokumen ini adalah HTML.
    <html lang="id"> → Elemen utama HTML, dengan bahasa Indonesia sebagai default.
    <head> → Bagian kepala halaman (tidak tampil di layar).
    <meta charset="UTF-8"> → Agar halaman mendukung karakter unik dan emoji.
    <meta viewport> → Membuat tampilan responsif di perangkat mobile.
    <title> → Judul halaman di tab browser.
    <body> → Bagian utama yang berisi konten halaman.
    <h1> → Judul utama halaman.
  5. Simpan dan Buka File
    Simpan file dengan nama index.html, lalu buka dengan cara klik dua kali. Pastikan nama file bukan index.html.txt.
  6. Tag HTML Penting Lainnya
    <h1> - <h6> → Judul (semakin besar angka, semakin kecil ukurannya).
    <p> → Paragraf.
    <style> → Mengatur tampilan dengan CSS (di dalam <head>).
    <img> → Menampilkan gambar.
    <a> → Membuat tautan (link).
    <ul> → List tidak berurutan.
    <ol> → List berurutan.
    <li> → Item dalam list.
    <br> → Baris baru.
    <hr> → Garis horizontal.

  8. GoogieHost — googiehost.com

Untuk website bisnis atau website dengan performa tinggi, disarankan menggunakan hosting berbayar.

Apa Itu Domain?

Domain adalah alamat website agar dapat diakses dengan mudah oleh orang lain. Domain terdiri dari dua bagian utama:

  • Custom Domain

    Custom domain adalah alamat website khusus seperti google.com, archive.org, dan itch.io.

    1. .com
      Domain paling umum dan cocok untuk hampir semua jenis website.
    2. .id
      Domain Indonesia, termasuk .my.id, .web.id.
    3. .org
      Domain organisasi dan banyak digunakan oleh proyek open source.
  • Subdomain

    Subdomain adalah bagian depan dari domain, seperti www.situsanda.com atau blog.situsanda.com. Pada beberapa layanan hosting gratis, subdomain digunakan sebagai alamat website, misalnya situsanda.pages.dev.

Bagaimana Cara Memberikan Navigasi pada Halaman HTML?

Navigasi pada halaman HTML berfungsi untuk berpindah dari satu halaman ke halaman lain, atau menuju website lain. Navigasi dasar dapat dibuat menggunakan link dan tombol.

  • Link

    Link adalah cara paling umum untuk membuat navigasi. Kamu dapat menggunakan tag <a>. Contohnya, untuk berpindah ke halaman lain dalam satu website: 

    <a href="halaman.html">Halaman Lain</a>

    Jika ingin menuju ke website lain: 

    <a href="https://namadomain.com">Website Lain</a>
  • Tombol

    Selain link, navigasi juga dapat dibuat menggunakan tombol. Tombol biasanya dikombinasikan dengan JavaScript sederhana menggunakan atribut onclick.

    Contoh tombol untuk menuju halaman lain: 

    <button onclick="window.location.href='halaman-lain.html'">
    Klik!
</button>

    Contoh tombol untuk menuju website lain: 

    <button onclick="window.location.href='https://website.com'">
    Klik!
</button>

Tampilan link dan tombol dapat diatur menggunakan CSS agar terlihat lebih menarik dan konsisten dengan desain website. Itulah cara membuat navigasi dasar pada halaman HTML.

Cara Memperbagus Tampilan Website HTML dengan CSS

Untuk memperbagus tampilan website HTML, kamu dapat menggunakan CSS. CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak halaman website, seperti warna, ukuran teks, jarak, dan posisi elemen HTML. Salah satu cara paling dasar adalah dengan menuliskan CSS langsung di dalam tag <style> yang berada di dalam tag <head>. Berikut contoh penggunaannya: 


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>

    <style>
        /* Style untuk seluruh halaman */
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
            color: #333;
        }

        /* Style untuk judul */
        h1 {
            color: #4361ee;
            text-align: center;
        }

        /* Style untuk paragraf */
        p {
            line-height: 1.6;
            margin: 20px;
        }
    </style>
</head>

<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah paragraf pertama di halaman HTML saya.</p>
</body>
</html>

Contoh di atas menunjukkan penggunaan CSS paling dasar untuk mempercantik tampilan website HTML. Selanjutnya, berikut beberapa properti CSS dasar yang sering digunakan untuk mengatur tampilan website.

  • color: Mengatur warna teks. Contoh: black, red, blue.
  • background-color: Mengatur warna latar belakang. Contoh: black, white, pink.
  • font-family: Mengatur jenis font. Contoh: Arial, Times New Roman, Courier New.
  • font-size: Mengatur ukuran teks. Contoh: 15px.
  • text-align: Mengatur perataan teks. Contoh: left, center, right.
  • margin: Mengatur jarak di luar elemen. Contoh: 10px, 20px.
  • padding: Mengatur jarak di dalam elemen. Contoh: 10px, 20px.
  • justify-self: Mengatur posisi satu elemen tertentu (pada grid/flex). Contoh: center, start, end.
  • justify-content: Mengatur posisi seluruh elemen di dalam container. Contoh: center, start, end.
  • justify-items: Mengatur posisi elemen di dalam ruang masing-masing. Contoh: center, start, end.

Untuk menerapkan properti CSS tersebut ke elemen HTML, cukup tuliskan nama elemen lalu tambahkan properti CSS di dalam tag <style>. Contohnya, jika ingin mengatur tampilan elemen <body>: 

body {
    /* warna teks */
    color: black;

    /* warna latar belakang */
    background-color: white;

    /* jenis font */
    font-family: Arial, sans-serif;
}

Itulah beberapa properti CSS dasar yang penting untuk memperbagus tampilan website HTML, beserta contoh penerapannya. Semoga bermanfaat! Cara Menggunakan AI untuk Membantu Pemrograman

Saat ini, banyak programmer memanfaatkan teknologi AI untuk membantu proses pengembangan proyek. Namun, tidak sedikit juga—terutama pemula—yang keliru dalam menggunakan AI untuk coding. Oleh karena itu, artikel ini akan membahas cara memakai AI untuk membantu pemrograman dengan baik dan benar.


1. Gunakan Prompt dengan Bahasa yang Jelas dan Mendetail

Prompt berperan penting dalam menentukan hasil yang diberikan oleh AI. Jika prompt terlalu singkat atau tidak jelas, kode yang dihasilkan berpotensi error atau tidak sesuai dengan ekspektasi. Semakin jelas prompt yang diberikan, semakin baik pula hasilnya.

Berikut contoh prompt AI yang baik, misalnya untuk membuat website profil pribadi:


"Saya ingin membuat website profil pribadi dengan tema terang (light theme), font modern, dan animasi sederhana. Website ini memiliki konten sebagai berikut:

1. ...

2. ...

... (dan seterusnya)

Tolong berikan struktur folder dan file, beserta potongan kode (snippet). Sertakan penjelasan pada setiap bagian. Jangan gunakan framework tertentu."


Sebaliknya, berikut contoh prompt yang kurang tepat:


"Buatkan saya sebuah website pribadi"


Hal-hal penting yang perlu diperhatikan saat membuat prompt yang baik:

  • Konteks yang jelas
  • Detail kebutuhan
  • Batasan atau aturan

2. Gunakan Metode "Copy – Paste – Modifikasi"

Metode ini berarti menyalin kode yang dihasilkan oleh AI, menempelkannya ke dalam proyek, lalu memodifikasinya sesuai kebutuhan. Cara ini tergolong sehat dalam penggunaan AI untuk coding.

AI membantu menghemat waktu dalam menulis kode, sementara kita tetap belajar dengan memahami dan mengubah hasilnya. Dengan begitu, kita tidak terjebak pada pola "copy–paste–selesai" tanpa memahami kode yang digunakan.

Mengapa Ukuran File HTML Bisa Membesar?

Ukuran file HTML bisa membesar karena berisi banyak karakter, seperti tag yang berlebihan, spasi dan baris kosong, komentar, JavaScript inline, serta CSS inline. Selain itu, penggunaan framework juga dapat menambah ukuran file website.

Jika kamu menggunakan HTML murni tanpa framework, ada beberapa cara sederhana untuk mengurangi ukuran file HTML.


1. Gunakan JavaScript dan CSS Eksternal

JavaScript dan CSS eksternal adalah kode yang ditulis di file terpisah, seperti .js untuk JavaScript dan .css untuk CSS. Berbeda dengan inline (kode langsung di dalam HTML), metode ini membuat file HTML menjadi lebih ringan dan rapi, terutama jika kodenya kompleks.

  • Buat file yang diperlukan, misalnya index.js atau index.css.
  • Salin kode dari tag <script> (JavaScript) atau <style> (CSS) tanpa menyertakan tag-nya.
  • Tempelkan kode tersebut ke file eksternal yang sesuai.

    Contoh: jika JavaScript inline berisi console.log('Halo dunia!');, maka isi file index.js menjadi: 

    console.log('Halo dunia!');
  • Panggil file eksternal ke dalam file HTML.

    Untuk memanggil file index.js: 

    <script src="index.js"></script>

    Untuk memanggil file index.css: 

    <link rel="stylesheet" href="index.css">

Penting: Pastikan path (lokasi file) sudah benar. Jika file berada di folder src/, maka gunakan src="src/namafile.js" atau href="src/namafile.css".


2. Minify File HTML

Minify HTML adalah proses menghapus spasi berlebih, baris kosong, dan komentar agar ukuran file menjadi lebih kecil.

  • Buat dua file: satu untuk development dan satu untuk deploy.

    File development berisi kode yang rapi dan mudah dibaca, misalnya index-dev.html. File deploy adalah hasil minify yang siap diunggah ke hosting, biasanya bernama index.html.

  • Minify file development menggunakan tools minifier.

    Kamu bisa menggunakan tools online seperti Toptal HTML Minifier . Setelah proses selesai, salin hasil minify ke file deploy, lalu uji di browser atau localhost untuk memastikan tidak ada error.
