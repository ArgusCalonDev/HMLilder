Dokumentasi

Component Library adalah sebuah tool di HMlilder yang berisi kumpulan template komponen siap pakai untuk website.

Step by step:

  • Buka Component Library di HMLilder
  • Pilih jenis komponen yang diperlukan
  • Lihat preview
  • Copy dan paste ke halaman HTML kamu
  • Modifikasi sesuai kebutuhan

(Opsional) Kamu bisa mendukung pengembangan HMLilder dengan membeli template website. Jika kamu membutuhkan template website premium, kamu dapat membantu tanpa mengeluarkan biaya tambahan.
