Dokumentasi Quick Accessibility Checker

Tool ini memeriksa 4 hal kunci: img tanpa alt, button tanpa teks, form label, dan kontras basic (inline style).

  1. Tempel HTML pada textarea.
  2. Klik "Jalankan Check".
  3. Lihat hasil di panel Analisis.

Hasil berupa daftar issue; jika kosong, muncul pesan sehat.
