Dokumentasi Inline Style to CSS Class

Tool ini mengonversi elemen HTML dengan atribut inline style menjadi class CSS dan menghapus style inline di output HTML.

Langkah singkat:

  1. Tempelkan HTML berisi inline style
  2. Klik "Convert ke CSS class"
  3. Gunakan hasil HTML dan CSS yang ditampilkan

Contoh input: <div class="box" style="color: red; padding: 10px">...

Contoh output CSS: .box { color: red; padding: 10px }
