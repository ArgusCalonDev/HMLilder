Dokumentasi Class Name Generator

Class Name Generator adalah tool untuk membantu membuat nama kelas CSS yang konsisten dari input nama komponen dan modifier.

Step by step:

  1. Buka Class Name Generator
  2. Masukkan jenis komponen (contoh: card, navbar, hero)
  3. Masukkan modifier/variasi (opsional)
  4. Klik "Generate Names"
  5. Salin hasil dari BEM atau utility naming

Fitur: output BEM style (`block__element`, `block--modifier`) dan utility style (`bg-...`, `text-...`, `p-...`).

Perlu tambahan: gunakan output tersebut sebagai acuan struktur kelas CSS di proyek kamu agar lebih terstandarisasi.
