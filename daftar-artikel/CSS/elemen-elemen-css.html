Elemen dan Properti Penting dalam CSS

Dalam CSS, terdapat banyak properti (sering disebut elemen CSS oleh pemula) yang berfungsi untuk mengatur tata letak, posisi, dan perilaku elemen HTML. Artikel ini membahas beberapa properti CSS yang paling sering digunakan, terutama untuk layout modern.

display

Properti display menentukan bagaimana sebuah elemen ditampilkan di halaman. Beberapa nilai yang sering digunakan adalah block, inline, inline-block, dan flex. 


.container {
  display: flex;
}

Saat menggunakan display: flex, elemen anak di dalamnya dapat diatur dengan lebih fleksibel menggunakan properti lain seperti justify-content dan align-items.

justify-content

justify-content digunakan untuk mengatur posisi elemen secara horizontal (sumbu utama) di dalam container flex. 


.container {
  display: flex;
  justify-content: center;
}

Beberapa nilai umum dari justify-content antara lain: flex-start, center, flex-end, space-between, dan space-around.

align-items

align-items digunakan untuk mengatur posisi elemen secara vertikal (sumbu silang) di dalam container flex. 


.container {
  display: flex;
  align-items: center;
}

Properti ini sangat berguna untuk membuat elemen berada di tengah secara vertikal tanpa trik CSS yang rumit.

gap

gap digunakan untuk memberi jarak antar elemen di dalam layout flex atau grid tanpa perlu margin manual. 


.container {
  display: flex;
  gap: 20px;
}

Penggunaan gap membuat kode CSS lebih rapi dan mudah dibaca, terutama pada layout modern.

position

Properti position mengatur cara sebuah elemen diposisikan. Nilai yang sering digunakan adalah static, relative, absolute, dan fixed. 


.box {
  position: relative;
  top: 10px;
}

Dengan position, kamu bisa mengatur elemen secara presisi sesuai kebutuhan layout.

overflow

overflow menentukan apa yang terjadi jika konten melebihi ukuran elemen. 


.box {
  overflow: auto;
}

Nilai yang sering digunakan adalah hidden, scroll, dan auto.

Kesimpulan

Properti-properti CSS di atas adalah fondasi penting dalam membangun layout modern. Dengan memahami fungsi tiap properti, kamu bisa menyusun tampilan website yang rapi, fleksibel, dan mudah dikembangkan dalam jangka panjang.
