Memperindah HTML Dengan CSS

Memperindah HTML dengan CSS - Pengertian CSS

CSS digunakan untuk mengatur tampilan visual website. HTML yang baik tanpa CSS akan terlihat polos, dan CSS yang baik tanpa HTML yang rapi akan sulit dirawat.

Apa Itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. CSS merupakan bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen HTML di dalam sebuah halaman website.

Jika HTML berfungsi sebagai struktur atau kerangka bangunan, maka CSS berfungsi sebagai desainnya. Dengan CSS, kamu bisa mengatur warna, ukuran teks, jenis font, jarak antar elemen, hingga tata letak halaman.

Hubungan HTML dan CSS

HTML dan CSS bekerja saling melengkapi. HTML menyusun struktur konten, sedangkan CSS memperindah dan mengatur tampilannya.

Pemisahan antara struktur (HTML) dan tampilan (CSS) membuat kode lebih rapi, mudah dibaca, serta lebih mudah dikembangkan dalam jangka panjang. Inilah alasan mengapa hampir semua website modern menggunakan CSS.

Fungsi Utama CSS

  • Mengatur warna teks dan latar belakang
  • Mengatur ukuran dan jenis font
  • Mengatur jarak antar elemen (margin dan padding)
  • Mengatur posisi dan tata letak (layout)
  • Membuat tampilan responsif untuk berbagai ukuran layar

Cara Menambahkan CSS ke HTML

Secara umum, CSS dapat ditambahkan ke dalam HTML dengan tiga cara:

1. Inline CSS

CSS ditulis langsung pada atribut elemen HTML. Cara ini biasanya digunakan untuk perubahan kecil, tetapi kurang efektif untuk project yang lebih besar.

2. Internal CSS

CSS ditulis di dalam tag <style> pada bagian <head>. Cocok untuk website sederhana yang hanya memiliki satu halaman.

3. External CSS

CSS ditulis dalam file terpisah dengan ekstensi .css, kemudian dihubungkan ke file HTML menggunakan tag <link>. Metode ini paling direkomendasikan karena membuat struktur kode lebih bersih, terorganisir, dan profesional.

Untuk jangka panjang, CSS sebaiknya dipisahkan ke file eksternal agar kode lebih bersih dan mudah dikelola.

Mengapa CSS Penting?

Tanpa CSS, website hanya akan menampilkan struktur dasar dengan tampilan standar dari browser. Dengan CSS, kamu bisa menciptakan desain yang lebih menarik, konsisten, dan nyaman digunakan oleh pengunjung.

Memahami CSS adalah langkah penting setelah menguasai HTML, karena kombinasi keduanya menjadi fondasi utama dalam pengembangan website modern.
