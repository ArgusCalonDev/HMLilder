JavaScript

JavaScript Dasar untuk Pemula

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif. Jika HTML berfungsi sebagai struktur dan CSS sebagai tampilan, maka JavaScript berperan sebagai logika dan perilaku.

Dengan JavaScript, kita bisa merespons klik pengguna, memproses data, menampilkan informasi secara dinamis, dan membangun aplikasi web sederhana langsung di browser.

Cara Menulis JavaScript

JavaScript dapat ditulis langsung di dalam file HTML menggunakan tag <script> atau dipisahkan ke dalam file berekstensi .js. 


console.log("Halo, JavaScript!");
// output: "Halo, Javascript!"

Kode di atas akan menampilkan teks ke dalam console browser, bukan ke halaman website. Untuk melihat output, klik tombol f12 pada keyboard, kemudian pilih tab "Console."

Variabel

Variabel digunakan untuk menyimpan data. JavaScript modern menyediakan let dan const sebagai cara utama mendeklarasikan variabel. 


let nama = "Ardhya";
const umur = 16;

console.log(nama);
console.log(umur);

// output: "Ardhya"
// output: 16

Gunakan const jika nilainya tidak berubah, dan let jika nilainya perlu diperbarui.

Tipe Data Dasar

JavaScript memiliki beberapa tipe data dasar seperti string, number, boolean, array, dan object. 


// string
const judul = "Belajar JavaScript";
// number 
const harga = 25000;
// boolean (true/false)
const aktif = true;

// array
const warna = ["merah", "biru", "hijau"];

// object
const user = {
  nama: "Adi",
  role: "Developer"
};

Percabangan

Percabangan digunakan untuk menjalankan kode berdasarkan kondisi tertentu. 


const nilai = 85;

if (nilai >= 75) {
  console.log("Lulus");
} else {
  console.log("Tidak lulus");
}

// jika nilai lebih dari 75, output menjadi "Lulus", namun jika nilai kurang dari 75, output menjadi "Tidak Lulus."

Perulangan

Perulangan digunakan untuk menjalankan kode yang sama berulang kali. 


for (let i = 1; i <= 5; i++) {
  console.log("Perulangan ke-" + i);
}

Fungsi

Fungsi digunakan untuk mengelompokkan kode agar dapat digunakan kembali dan membuat program lebih terstruktur. 


function sapa(nama) {
  return "Halo, " + nama;
}

console.log(sapa("Ardhya"));

// output: "Halo, Ardhya"

Komentar

Komentar digunakan untuk menandakan suatu bagian dalam kode JavaScript. Hal ini biasanya digunakan untuk kolaborasi, agar kolaborator bisa memahami bagian kode yang ditulis. Dalam JavaScript, komentar diawali dengan "//". Sebagai contoh: 


console.log("Halo Dunia!");
// ini adalah komentar!

Kesimpulan

JavaScript adalah fondasi penting dalam pengembangan web modern. Dengan memahami dasar-dasarnya, kamu sudah siap melangkah ke tahap berikutnya seperti manipulasi DOM dan event handling.
