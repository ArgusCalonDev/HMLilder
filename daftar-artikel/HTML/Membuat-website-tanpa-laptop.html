Membuat Website HTML Menggunakan HP Android Tanpa Laptop

Membuat website tidak selalu membutuhkan laptop atau komputer dengan spesifikasi tinggi. Dengan HP Android dan koneksi internet yang stabil, kamu sudah bisa membuat dan mengunggah website sederhana berbasis HTML. Berikut langkah-langkahnya dari awal hingga website online.

1. Install Aplikasi Code Editor di HP

Langkah pertama adalah memasang aplikasi editor kode.

Beberapa aplikasi yang bisa kamu gunakan di Android:

  • Acode
  • Spck Editor
  • QuickEdit

Cari di Play Store, lalu install salah satu. Setelah terpasang, kamu sudah siap menulis kode HTML.

Alternatif: Kamu dapat menggunakan tool Online HTML Editor kami tanpa menginstall aplikasi pada perangkat.

2. Membuat file HTML pertama

  1. Buka aplikasi editor kode
  2. Buat file baru dengan nama: index.html
  3. Isi dengan struktur dasar HTML berikut:
    4. 
<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Website ini dibuat hanya dengan HP Android.</p>
</body>
</html>
  4. Simpan file tersebut.

Untuk melihat hasilnya:

  • Buka file melalui file manager
  • Pilih "buka dengan browser"

Jika berhasil, halaman sederhana akan muncul di browser.

3. Menambahkan CSS (opsional)

Untuk mempercantik tampilan halaman, kamu dapat menambahkan css langsung dalam file HTML tersebut. Kamu dapat melihat panduan dasar CSS di sini.

4. Menyiapkan Website Agar Bisa Online

Agar website bisa diakses orang lain, kamu perlu mengunggahnya ke layanan hosting. Untuk pemula, berikut beberapa hosting gratis yang bisa digunakan:

  • Cloudflare Pages
  • Netlify
  • Vercel
  • Github pages

5. Cara upload ke hosting gratis

A. Cloudflare Pages

  1. Buat akun di Cloudflare
  2. Masuk ke menu Compute > Workers And Pages
  3. Klik tombol "Create Application" di pojok kanan atas
  4. Di Bagian bawah, pilih "Looking to deploy Pages? Get started"
  5. Upload file HTML atau file statis milikmu dalam bentuk ZIP
  6. Link dan projek akan otomatis dibuat
  7. Website langsung online dengan domain gratis dari Cloudflare.

B. Netlify

  1. Buat akun di Netlify.
  2. Login ke dashboard.
  3. Pilih “Add new site”.
  4. Gunakan opsi drag-and-drop.
  5. Upload file HTML kamu.
  6. Website langsung online dengan domain gratis dari Netlify.

C. Vercel

  1. Buat akun di Vercel.
  2. Hubungkan ke repository Git (biasanya dari GitHub).
  3. Upload project HTML.
  4. Klik deploy.
  5. Tunggu proses selesai, lalu dapatkan link website.

D. Github Pages

  1. Buat akun di GitHub.
  2. Buat repository baru.
  3. Upload file index.html.
  4. Masuk ke Settings > Pages.
  5. Aktifkan GitHub Pages.
  6. Website akan tersedia di domain username.github.io/repository

Penutup

Membuat website HTML menggunakan HP Android tanpa laptop sangat memungkinkan, terutama untuk website sederhana. Dengan editor ringan dan hosting gratis, kamu sudah bisa memulai perjalanan sebagai web developer.

Jika ingin panduan lebih lengkap tentang masing-masing layanan hosting, kamu bisa mencari tutorial detailnya di internet sesuai platform yang dipilih.

Yang terpenting adalah mulai dari sekarang. Tidak perlu menunggu perangkat mahal untuk belajar membuat website.
