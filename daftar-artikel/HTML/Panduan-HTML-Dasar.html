Panduan HTML Dasar

Panduan membuat halaman HTML Dasar

Membuat website HTML tidak membutuhkan software mahal atau alat khusus. Yang dibutuhkan hanyalah editor teks, browser, dan pemahaman dasar tentang struktur dokumen HTML. Fokus utama di tahap awal adalah keterbacaan dan kerapian struktur, bukan tampilan visual.

Langkah Dasar Membuat Website HTML

  1. Pilih Editor Teks
    Kamu bisa menggunakan editor bawaan seperti Notepad atau TextEdit, namun editor kode seperti Visual Studio Code lebih disarankan karena membantu penulisan kode dengan rapi dan minim kesalahan.
  2. Buat File HTML Baru
    Simpan file dengan ekstensi .html. Untuk halaman utama website, gunakan nama index.html karena ini adalah standar yang dikenali oleh hampir semua server.
  3. Tulis Struktur Dasar HTML 
    
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
</body>
</html>
  4. Makna Setiap Bagian
    HTML bekerja seperti dokumen formal. Setiap bagian memiliki peran yang jelas:

    <!DOCTYPE html> → Menandakan standar HTML modern.
    <html lang="id"> → Bahasa utama konten halaman.
    <head> → Informasi halaman untuk browser dan mesin pencari.
    <meta charset="UTF-8"> → Mendukung karakter internasional.
    <meta viewport> → Kunci tampilan responsif.
    <title> → Identitas halaman di tab browser.
    <body> → Semua konten yang terlihat pengguna.
  5. Simpan dan Jalankan
    Buka file HTML langsung di browser. Pastikan ekstensi file benar dan tidak ganda seperti .html.txt.

Catatan penting: HTML yang rapi sejak awal akan sangat membantu ketika website berkembang menjadi besar. Hindari kebiasaan asal menulis tag tanpa struktur.
