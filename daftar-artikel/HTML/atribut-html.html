Atribut HTML

Dalam HTML, sebuah tag tidak hanya berisi nama tag saja. Kita juga bisa menambahkan atribut untuk memberikan informasi tambahan atau mengatur perilaku dari tag tersebut.

Atribut biasanya ditulis di dalam tag pembuka. 


<img src="kucing.jpg" alt="kucing lucu">

Pada contoh di atas:

  • src menentukan sumber gambar
  • alt berisi teks alternatif jika gambar gagal dimuat

Dengan kata lain, atribut membuat sebuah tag HTML menjadi lebih informatif dan lebih terkontrol.

Struktur Penulisan Atribut

Penulisan atribut HTML memiliki pola yang sederhana. 


<tag namaAtribut="nilai">

Contoh:


<a href="https://example.com">Kunjungi Website</a>
  • href adalah nama atribut
  • https://example.com adalah nilai atribut

Atribut selalu berada di tag pembuka dan biasanya menggunakan tanda kutip untuk menuliskan nilainya.

Contoh Atribut yang Sering Digunakan

href

Atribut ini digunakan pada tag <a> untuk menentukan alamat tujuan link. 


<a href="https://google.com">Kunjungi Google</a>

src

Atribut src digunakan untuk menentukan sumber file, biasanya pada gambar atau media lainnya. 


<img src="foto.jpg">

alt

Atribut alt memberikan teks alternatif pada gambar. 


<img src="kucing.jpg" alt="kucing sedang tidur">
  • ditampilkan jika gambar gagal dimuat
  • membantu aksesibilitas
  • baik untuk SEO

class

Atribut class digunakan untuk memberi kelas pada sebuah elemen. 


<div class="card"></div>

Class biasanya digunakan untuk styling dengan CSS atau seleksi elemen dengan JavaScript. 


<div class="card shadow rounded"></div>

id

Atribut id digunakan untuk memberikan identitas unik pada sebuah elemen. 


<section id="hero"></section>

Berbeda dengan class, sebuah id seharusnya hanya digunakan satu kali dalam satu halaman.

Boolean Attribute

Beberapa atribut HTML tidak membutuhkan nilai. Atribut seperti ini disebut boolean attribute. 


<input type="checkbox" checked>

Atribut checked membuat checkbox langsung aktif secara default.

Contoh boolean attribute lainnya:

  • checked
  • disabled
  • required
  • autoplay
  • selected

<input type="text" required>

Global Attributes

HTML memiliki beberapa atribut yang bisa digunakan pada hampir semua tag. Atribut ini disebut global attributes.

  • class
  • id
  • style
  • title
  • hidden
  • data-*

Contoh penggunaan atribut title:


<p title="Ini tooltip">Arahkan mouse ke sini</p>

Custom Data Attribute (data-*)

HTML menyediakan atribut khusus untuk menyimpan data tambahan di dalam elemen HTML. 


<div data-user-id="123"></div>

Atribut ini sering digunakan bersama JavaScript. 


element.dataset.userId

Tips Penulisan Atribut

Gunakan tanda kutip saat menulis nilai atribut.


<input type="text">

Gunakan class untuk styling dan gunakan id untuk elemen yang unik dalam halaman.

Selalu tambahkan atribut alt pada gambar untuk meningkatkan aksesibilitas.

Kesalahan Umum Pemula

Salah menempatkan atribut:


<p> class="text"

Yang benar:


<p class="text">

Lupa tanda kutip:


href=https://google.com

Yang benar:


href="https://google.com"

Kesimpulan

Atribut HTML digunakan untuk memberikan informasi tambahan pada sebuah tag. Dengan atribut, kita bisa mengatur link, gambar, form input, styling, hingga data yang digunakan oleh JavaScript.

Memahami atribut HTML adalah langkah penting sebelum mempelajari CSS dan manipulasi DOM dengan JavaScript.
