Kenapa CSS Saya Tidak Berfungsi?

Pendahuluan

Salah satu masalah paling umum yang dialami pemula adalah CSS yang tidak bereaksi sama sekali. Sudah menulis style, sudah save file, tapi tampilan tetap polos seperti tanpa desain. Masalah ini biasanya bukan karena CSS sulit, melainkan karena kesalahan kecil yang sering tidak disadari.

Di artikel ini kita akan membahas penyebab paling umum dan cara mengeceknya secara sistematis.

Penyebab Umum CSS Tidak Berfungsi

1. File CSS Tidak Terhubung dengan Benar

Pastikan di dalam <head> ada kode seperti ini: 


<link rel="stylesheet" href="style.css">

Periksa:

  • Nama file benar (tidak typo)
  • Lokasi sesuai dengan path
  • Ekstensi benar (.css)

Jika file berada di folder lain, gunakan path yang sesuai, misalnya: 


<link rel="stylesheet" href="css/style.css">

2. Path file salah

Jika file HTML berada di folder berbeda dari CSS, kamu mungkin perlu menambahkan ../ untuk naik satu folder.

3. CSS Tertimpa (Override) oleh Style Lain

CSS bisa kalah oleh selector yang lebih spesifik.

Contoh:


p { 
    color: red;
}
.container p { 
    color: blue; 
}

Paragraf didalam class container akan berwarna biru karena selector lebih spesifik.

4. Cache browser

Kadang perubahan tidak muncul karena browser menyimpan versi lama. Coba:

  • Hard refresh halaman(ctrl + f5)
  • Clear cache browser
  • Atau buka di mode incognito

5. Typo atau kurang tanda kurung

Contoh kesalahan umum: 


p {
    color: red

Seharusnya:


p {
    color: red;
}

Kesimpulan

Jika CSS tidak berfungsi, jangan panik. Periksa secara berurutan: koneksi file, path, typo, specificity, dan cache. Sebagian besar masalah hanya kesalahan kecil yang mudah diperbaiki jika dicek dengan teliti.
