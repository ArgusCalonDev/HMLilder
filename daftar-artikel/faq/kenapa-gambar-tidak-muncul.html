Kenapa Gambar Tidak Muncul di HTML?

Pendahuluan

Salah satu masalah paling umum saat belajar HTML adalah gambar yang tidak tampil. Biasanya yang terlihat hanya ikon gambar rusak atau ruang kosong di halaman. Penyebabnya hampir selalu sederhana, seperti kesalahan penulisan path atau nama file.

Penyebab Umum Gambar Tidak Muncul

1. Path File Salah

Pastikan atribut src mengarah ke lokasi file yang benar.


<img src="images/foto.jpg" alt="Contoh">

Perhatikan huruf besar dan kecil karena beberapa server bersifat case-sensitive.

2. Nama File atau Ekstensi Salah

Pastikan nama file dan ekstensi sesuai, misalnya .jpg, .png, atau .webp. Kesalahan kecil seperti .JPG dan .jpg bisa membuat gambar tidak tampil.

3. File Tidak Ada di Folder Proyek

Pastikan file benar-benar berada di dalam folder proyek website, bukan hanya di folder download komputer.

4. CSS Menyembunyikan Gambar


img {
    display: none;
}

Periksa apakah ada style yang menyembunyikan gambar atau mengatur ukurannya menjadi 0.

Kesimpulan

Jika gambar tidak muncul, periksa path, nama file, dan struktur folder terlebih dahulu. Sebagian besar masalah terjadi karena kesalahan penulisan yang sangat kecil.
