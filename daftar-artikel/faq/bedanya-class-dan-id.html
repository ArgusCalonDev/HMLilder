Perbedaan Class dan ID di HTML

Pendahuluan

Dalam HTML, class dan id digunakan untuk memberikan identitas pada elemen. Keduanya sering digunakan dalam CSS dan JavaScript, tetapi memiliki fungsi dan aturan yang berbeda. Memahami perbedaannya penting agar struktur kode tetap rapi dan tidak bermasalah.

Apa Itu Class?

Class digunakan untuk menandai beberapa elemen sekaligus. Satu class bisa dipakai berkali-kali dalam satu halaman. 


<div class="box">Konten 1</div>
<div class="box">Konten 2</div>

Contoh CSS:


.box {
    background-color: lightblue;
}

Apa Itu ID?

ID digunakan untuk menandai satu elemen unik. Dalam satu halaman HTML, satu ID hanya boleh digunakan sekali. 


<div id="header">Header Website</div>

Contoh CSS:


#header {
    background-color: gray;
}

Perbedaan Utama Class dan ID

  • Jumlah penggunaan: Class bisa digunakan berkali-kali, ID hanya sekali.
  • Penulisan CSS: Class menggunakan tanda titik (.), ID menggunakan tanda pagar (#).
  • Spesifisitas CSS: ID memiliki prioritas lebih tinggi dibanding class.
  • JavaScript: ID sering digunakan untuk memilih satu elemen spesifik.

Kapan Menggunakan Class dan ID?

Gunakan class untuk styling berulang seperti tombol, kartu, atau layout. Gunakan id untuk elemen unik seperti header utama, navigasi, atau bagian tertentu yang perlu ditargetkan secara spesifik.

Kesimpulan

Class cocok untuk elemen yang berulang, sedangkan ID digunakan untuk elemen unik. Menggunakan keduanya dengan tepat akan membuat kode HTML lebih terstruktur dan mudah dikelola.
