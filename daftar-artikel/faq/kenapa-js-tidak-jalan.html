Kenapa JavaScript Saya Tidak Jalan?

Pendahuluan

Salah satu masalah paling sering dialami pemula adalah JavaScript yang tidak berjalan. Tombol diklik tapi tidak bereaksi, alert tidak muncul, atau fitur tidak berfungsi. Biasanya penyebabnya sederhana dan bisa diperbaiki dengan cepat.

Penyebab Umum JavaScript Tidak Berjalan

1. File JavaScript Tidak Terhubung

Pastikan file JS sudah dihubungkan dengan benar menggunakan tag <script>. 


<script src="script.js"></script>

Periksa kembali nama file dan path folder. Salah huruf saja bisa membuat file tidak terbaca.

2. Salah Penempatan Script

Jika script ditempatkan di bagian <head> tanpa atribut defer, kadang JavaScript dijalankan sebelum HTML selesai dimuat.

Solusi:


<script src="script.js" defer></script>

Atau letakkan sebelum tag </body>.

3. Ada Error di Console

Buka Developer Tools (F12) lalu cek tab Console. Jika ada error berwarna merah, JavaScript bisa berhenti berjalan.

4. DOM Belum Dimuat

Jika kamu memanggil elemen sebelum halaman selesai dimuat, elemen tersebut akan bernilai null.

Gunakan event berikut:


document.addEventListener("DOMContentLoaded", function() {
    // kode di sini
});

5. Salah Penulisan Kode (Syntax Error)

Kurang tanda titik koma, kurung, atau tanda kutip bisa menyebabkan script gagal dijalankan. JavaScript sangat sensitif terhadap kesalahan kecil.

Kesimpulan

Jika JavaScript tidak berjalan, periksa koneksi file, posisi script, cek console untuk error, dan pastikan DOM sudah dimuat sebelum menjalankan kode. Dengan debugging yang tepat, masalah ini biasanya bisa diselesaikan dengan cepat.
