Kenapa Link Tidak Bisa Diklik di HTML?

Pendahuluan

Link adalah elemen penting dalam sebuah website. Jika link tidak bisa diklik, tentu akan mengganggu navigasi pengguna. Masalah ini biasanya disebabkan oleh kesalahan kecil pada atribut HTML, CSS, atau JavaScript.

Penyebab Link Tidak Bisa Diklik

1. Atribut href Tidak Ada

Tanpa atribut href, link tidak akan aktif.


<a>Home</a>

Seharusnya:


<a href="index.html">Home</a>

2. Tertutup Elemen Lain

Elemen lain dengan position: absolute atau z-index tinggi dapat menutupi link. Gunakan Inspect Element untuk memastikan tidak ada elemen yang menimpanya.

3. CSS pointer-events


a {
    pointer-events: none;
}

Properti ini membuat elemen tidak bisa menerima klik.

4. Dicegah oleh JavaScript


event.preventDefault();

Jika digunakan pada event klik, link tidak akan berpindah halaman.

Kesimpulan

Jika link tidak bisa diklik, periksa atribut href, kemungkinan tertutup elemen lain, pengaturan CSS, serta script JavaScript yang mungkin mencegah aksi default.
