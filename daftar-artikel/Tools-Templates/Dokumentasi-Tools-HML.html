Dokumentasi Tools HMLilder

Page Builder HMLilder

Page Builder adalah salah satu tool pada HMLilder yang berfungsi sebagai pembuat halaman HTML sederhana.

    Cara Pakai:
  • Pilih template yang disediakan
  • Isi kolom "Nama Bisnis Anda" sesuai judul halaman kamu
  • Isi kolom "Tagline/Slogan"
  • Isi kolom "Deskripsi"
  • Isi Kolom dan opsi lainnya yang tersedia, sesuai kebutuhan kamu
  • Preview halaman berada di bagian "Live Preview"
  • Copy atau download HTML, dan modifikasi sesuai kebutuhan.

Component Generator HMLilder

Component Generator adalah salah satu tool pada HMLilder yang berfungsi sebagai pembuat potongan komponen website.

    Cara Pakai:
  • Pilih komponen yang disediakan
  • Lihat Preview
  • Lihat output kode HTML & CSS
  • Copy kode HTML pada tag <body> pada halaman kamu
  • Copy kode CSS pada tag <style> pada halaman kamu
  • Modifikasi sesuai kebutuhan

HTML Editor HMLilder

HTML Editor adalah salah satu tool pada HMLilder yang berfungsi sebagai editor HTML online sederhana.

    Cara Pakai:
  • Tulis kode HTML milikmu
  • Lihat preview
  • Download file

HTML Minifier HMLilder

HTML Minifier adalah salah satu tool pada HMLilder yang berfungsi untuk mengkompresi ukuran file HTML milikmu.

    Cara Pakai:
  • Paste seluruh kode HTML Milikmu
  • Klik "Minify HTML Sekarang!"
  • Lihat output
  • Copy manual dari kotak output

(Opsional) Kamu bisa mendukung pengembangan HMLilder dengan menggunakan layanan yang kami rekomendasikan. Jika kamu memang membutuhkan hosting atau domain, menggunakan link ini bisa membantu kami tanpa biaya tambahan.
