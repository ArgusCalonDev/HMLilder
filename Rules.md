# Rules Kolaborasi HMLilder

Dokumen ini berisi peraturan dasar yang wajib dipahami dan dipatuhi oleh setiap kolaborator dalam pengembangan HMLilder.

## 1. Kemampuan Dasar
Kamu wajib menguasai **HTML, CSS, dan JavaScript vanilla (tanpa framework)** minimal di tingkat dasar. Pemahaman ini penting agar proses kolaborasi berjalan lancar dan kode yang dihasilkan tetap konsisten.

## 2. Penggunaan GitHub
Setiap kolaborator wajib menggunakan GitHub sebagai media kolaborasi.  
Panduan lengkap mengenai alur kerja kolaborasi dapat kamu baca pada file **How-To-Collab.md**.

## 3. Modifikasi File
Kamu hanya diperbolehkan memodifikasi file sesuai dengan tugas atau arahan yang diberikan oleh **saya**.  
Tujuannya agar pekerjaan tetap terarah dan tidak saling bertabrakan dengan bagian lain yang sedang dikerjakan.

## 4. Larangan Modifikasi File Build
Dilarang memodifikasi file yang sudah dalam kondisi **minify** atau merupakan **hasil build**.  
Perubahan hanya boleh dilakukan pada file sumber yang masih dalam bentuk asli dan mudah dibaca.

## 5. Penggunaan AI
Penggunaan AI diperbolehkan, dengan catatan:
- Kamu memahami kode yang dihasilkan oleh AI.
- Kamu tidak melakukan copy paste secara membabi buta tanpa mengerti fungsinya.

AI digunakan sebagai alat bantu, bukan sebagai pengganti pemahaman.

## 6. Komunikasi Kendala
Jika kamu memiliki kesibukan, kendala, atau kondisi tertentu yang membuat kamu tidak dapat mengerjakan tugas, mohon sampaikan secara jujur dan jelas.  
Komunikasi yang baik jauh lebih dihargai daripada menghilang tanpa kabar.

## 7. Sistem Kerja Non-Komersial
HMLilder adalah proyek **non-komersial** yang dikembangkan secara **santai, berbasis hobi, dan tanpa deadline**.  
Proyek ini **tidak bertujuan mencari keuntungan finansial**, tidak memiliki target bisnis, dan tidak menuntut produktivitas layaknya proyek profesional.

Setiap kolaborator berkontribusi secara sukarela sesuai waktu luang, minat, dan kapasitas masing-masing.  
**Tidak ada bayaran, kontrak kerja, maupun kewajiban profesional** dalam proyek ini.

Fokus utama HMLilder adalah pembelajaran, eksplorasi, berbagi pengetahuan, dan menikmati proses pengembangan bersama.

---

Dengan mengikuti peraturan ini, diharapkan kolaborasi HMLilder dapat berjalan dengan nyaman, rapi, dan saling menghargai.
