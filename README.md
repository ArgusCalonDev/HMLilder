# HMLilder
**HMLilder** adalah platform yang menyediakan **tools dan artikel** untuk mempermudah dan mempercepat pembuatan website Kamu. Tools yang tersedia:

1. Page Builder
Page Builder membantu kamu membuat suatu halaman html hanya dengan mengisi kolom yang ada, melihat preview, lalu download file html akhir, atau copy paste kode, dan kemudian kamu bisa memodifikasinya di editor kode.

2. Component Generator
Component generator membantu membuat komponen website HTML berupa snippet (potongan) siap pakai untuk website HTML kamu. Pilih jenis komponen, lihat preview, copy hasil, paste di file HTML kamu, dan modifikasi sesuka hati.

3. Online HTML Editor
Online HTML Editor memungkinkan kamu mengedit kode HTML kamu secara online, tanpa menginstal editor kode di komputer kamu. 

Untuk kolaborasi HMLilder, kamu dapat membaca file **How-To-Collab.md** untuk panduan. Baca juga file **Rules.md** untuk peraturan dalam kolaborasi.
