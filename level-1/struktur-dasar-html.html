Struktur Dasar HTML (html, head, body)

Dalam sebuah file HTML, ada tiga bagian utama yang hampir selalu digunakan, yaitu html, head, dan body. Ketiga bagian ini membantu browser memahami struktur halaman website.

Berikut contoh struktur sederhananya: 


<html>

<head>
    <title>Website Saya</title>
</head>

<body>
    <h1>Halo Dunia</h1>
    <p>Ini halaman pertama saya.</p>
</body>

</html>

Tag html

Tag <html> adalah elemen utama yang membungkus seluruh isi halaman website. Semua kode HTML biasanya ditulis di dalam tag ini.

Tag head

Tag <head> berisi informasi tentang halaman website, seperti judul halaman, pengaturan meta, atau file CSS.

Konten di dalam head biasanya tidak terlihat langsung di halaman.

Tag body

Tag <body> berisi semua konten yang ditampilkan di halaman website, seperti teks, gambar, link, dan elemen lainnya.

Semua yang ingin dilihat oleh pengunjung website biasanya ditulis di dalam bagian body.
