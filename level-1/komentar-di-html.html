Komentar di HTML

Saat menulis kode HTML, kadang kita ingin menambahkan catatan di dalam kode. Catatan ini disebut komentar.

Komentar tidak akan ditampilkan di halaman website. Browser akan mengabaikan bagian ini, sehingga hanya berfungsi sebagai penjelasan di dalam kode.

Contoh Komentar HTML


<!-- Ini adalah komentar di HTML -->

Semua teks di antara <!-- dan --> akan dianggap sebagai komentar.

Contoh Penggunaan


<!-- Judul halaman -->
<h1>Website Saya</h1>

<!-- Paragraf pembuka -->
<p>Selamat datang di website saya.</p>

Komentar sering digunakan untuk memberi penjelasan pada bagian kode, sehingga kode lebih mudah dipahami ketika dibaca kembali.
