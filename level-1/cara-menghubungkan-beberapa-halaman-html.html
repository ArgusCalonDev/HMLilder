Cara Menghubungkan Beberapa Halaman HTML

Sebuah website biasanya tidak hanya memiliki satu halaman. Website bisa memiliki beberapa halaman seperti halaman utama, halaman artikel, atau halaman tentang.

Untuk menghubungkan halaman-halaman tersebut, kita bisa menggunakan link dengan tag <a>.

Contoh Struktur File

Misalnya kita memiliki beberapa file HTML seperti ini: 


index.html
about.html
contact.html

Menghubungkan Halaman

Kita bisa membuat link dari halaman utama ke halaman lainnya. 


<a href="about.html">Halaman Tentang</a>
<a href="contact.html">Halaman Kontak</a>

Ketika link tersebut diklik, browser akan membuka file HTML yang sesuai dengan alamat di dalam atribut href.

Dengan cara ini, kamu bisa membuat beberapa halaman yang saling terhubung dan membentuk sebuah website.
