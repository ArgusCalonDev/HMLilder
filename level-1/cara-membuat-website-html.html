Cara Membuat Website HTML Pertama

Membuat website sebenarnya tidak serumit yang dibayangkan. Bahkan dengan satu file HTML saja, kamu sudah bisa membuat sebuah halaman website sederhana.

Di artikel ini kita akan mencoba membuat halaman website pertama yang sangat sederhana, lalu membukanya langsung di browser.

1. Buat File HTML

Pertama, buat sebuah file baru di komputer kamu. Kamu bisa membuatnya menggunakan text editor seperti Visual Studio Code atau Notepad++ jika menggunakan laptop, atau menggunakan text editor seperti Acode atau TrebEdit jika menggunakan hp Android.

Beri nama file tersebut misalnya index.html. Ekstensi .html memberi tahu browser bahwa file tersebut adalah halaman website.

2. Tulis Kode HTML Sederhana

Setelah file dibuat, tulis kode HTML sederhana seperti berikut: 


<h1>Halo Dunia!</h1>
<p>Ini adalah website pertama saya.</p>

Kode di atas membuat sebuah judul besar dan satu paragraf teks di halaman website.

3. Buka di Browser

Setelah menyimpan file HTML, kamu bisa langsung membukanya di browser. Caranya cukup klik dua kali file index.html tersebut.

Browser akan membaca kode HTML dan menampilkannya sebagai halaman website.

Walaupun sangat sederhana, ini sebenarnya sudah termasuk sebuah website. Dari sini kamu bisa mulai menambahkan lebih banyak elemen HTML untuk membuat halaman yang lebih lengkap.
