Apa Itu Website?

Website adalah kumpulan halaman yang bisa diakses melalui internet menggunakan browser. Halaman-halaman ini biasanya berisi berbagai jenis informasi, seperti teks, gambar, video, atau bahkan aplikasi yang bisa kita gunakan langsung dari browser.



Contoh website yang sering kita gunakan sehari-hari adalah mesin pencari, media sosial, toko online, atau website berita. Saat kamu mengetik alamat website di browser (misalnya namasitus.com), browser akan meminta data ke server di internet, lalu server mengirimkan halaman website tersebut agar bisa ditampilkan di layar kamu.



Secara teknis, sebuah website biasanya dibuat menggunakan bahasa seperti HTML, CSS, dan JavaScript. HTML digunakan untuk membuat struktur halaman, CSS untuk mengatur tampilan, dan JavaScript untuk menambahkan interaksi.



Sederhananya, website adalah halaman digital di internet yang bisa dibuka oleh siapa saja melalui browser. Halaman ini bisa berisi informasi sederhana, sampai aplikasi web yang kompleks seperti email, editor dokumen, atau platform belajar online.
