Mengatur Ukuran dan Spasi dengan CSS

CSS juga bisa digunakan untuk mengatur ukuran elemen dan jarak antar elemen di halaman website. Dengan pengaturan ini, tampilan halaman bisa terlihat lebih rapi dan nyaman dibaca.

Mengatur Ukuran Teks

Untuk mengubah ukuran teks, kita bisa menggunakan properti font-size. 


p {
    font-size: 18px;
}

Kode di atas akan membuat ukuran teks paragraf menjadi 18 piksel.

Mengatur Spasi

CSS juga menyediakan properti untuk mengatur jarak antar elemen, seperti margin dan padding. 


div {
    margin: 20px;
    padding: 10px;
}

margin digunakan untuk mengatur jarak di luar elemen, sedangkan padding mengatur jarak di dalam elemen.

Dengan mengatur ukuran dan spasi menggunakan CSS, tampilan website bisa menjadi lebih terstruktur dan enak dilihat.
