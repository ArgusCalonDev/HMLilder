Bagaimana Cara Kerja Website Di Internet?

Saat kamu membuka sebuah website, sebenarnya browser dan server sedang saling berkomunikasi melalui internet.

Misalnya kamu mengetik alamat seperti contoh.com di browser. Setelah itu, browser akan mengirim permintaan ke server tempat website tersebut disimpan. Permintaan ini disebut request.

Server kemudian mengirimkan file website seperti HTML, CSS, dan JavaScript kembali ke browser. Proses ini disebut response.

Setelah menerima file tersebut, browser akan menyusunnya menjadi halaman website yang bisa kamu lihat di layar.

Sederhananya, prosesnya seperti ini:

  1. Kamu membuka alamat website di internet
  2. Browser meminta data ke server
  3. Server mengirim file website
  4. Browser menampilkan halaman website

Semua proses ini terjadi sangat cepat, biasanya hanya dalam beberapa detik.
