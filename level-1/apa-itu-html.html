Apa Itu HTML?

HTML adalah bahasa dasar yang digunakan untuk membuat halaman website. Hampir semua website di internet menggunakan HTML sebagai fondasinya.

HTML digunakan untuk menyusun struktur halaman, seperti judul, paragraf, gambar, link, dan berbagai elemen lainnya yang muncul di website.

HTML bekerja menggunakan tag. Tag adalah penanda yang memberi tahu browser bagaimana sebuah bagian halaman harus ditampilkan.

Contohnya seperti ini: 


<h1>Halo Dunia</h1>
<p>Ini adalah paragraf di website.</p>

Tag <h1> digunakan untuk membuat judul, sedangkan <p> digunakan untuk membuat paragraf.

Browser akan membaca kode HTML tersebut, lalu menampilkannya menjadi halaman website yang bisa kita lihat.

Sederhananya, HTML adalah bahasa yang digunakan untuk membangun struktur sebuah halaman website.
