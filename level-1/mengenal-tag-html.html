Mengenal Tag HTML

Saat menulis HTML, kamu akan sering melihat sesuatu yang disebut tag. Tag adalah bagian penting dalam HTML yang digunakan untuk memberi tahu browser bagaimana sebuah konten harus ditampilkan.

Tag biasanya ditulis menggunakan tanda kurung sudut, yaitu < >.

Contohnya seperti ini: 


<h1>Halo Dunia</h1>

Pada contoh di atas, <h1> adalah tag yang digunakan untuk membuat judul besar di halaman website.

Tag Pembuka dan Tag Penutup

Banyak tag HTML memiliki dua bagian, yaitu tag pembuka dan tag penutup. Tag penutup biasanya memiliki tanda garis miring /. 


<p>Ini adalah paragraf</p>

<p> adalah tag pembuka, sedangkan </p> adalah tag penutup. Semua teks di antara tag tersebut akan dianggap sebagai isi paragraf.

Dengan memahami tag HTML, kamu bisa mulai membuat berbagai elemen seperti judul, paragraf, gambar, link, dan lainnya di halaman website.
