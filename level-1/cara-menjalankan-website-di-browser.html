Cara Menjalankan Website di Browser

Setelah membuat file HTML, langkah berikutnya adalah menjalankan atau membuka website tersebut di browser. Dengan begitu kamu bisa melihat bagaimana halaman website ditampilkan.

Cara menjalankannya sebenarnya sangat sederhana. Kamu hanya perlu membuka file HTML menggunakan browser.

Menjalankan Website di Komputer atau Laptop

Jika kamu menggunakan komputer atau laptop, caranya cukup mudah.

  1. Buat atau simpan file HTML, misalnya index.html.
  2. Cari file tersebut di folder tempat kamu menyimpannya.
  3. Klik dua kali file tersebut.

Setelah itu file akan terbuka di browser, dan halaman website akan langsung ditampilkan.

Menjalankan Website di HP

Jika kamu membuat file HTML di HP, kamu juga bisa membukanya langsung di browser.

  1. Simpan file HTML di penyimpanan HP.
  2. Buka aplikasi file manager.
  3. Tap file .html yang ingin dibuka.

Browser di HP akan membuka file tersebut dan menampilkan halaman website seperti biasanya.

Dengan cara ini kamu bisa langsung melihat hasil website yang kamu buat tanpa perlu menguploadnya ke internet terlebih dahulu.
