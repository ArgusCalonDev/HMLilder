Membuat Link di HTML

Link digunakan untuk berpindah dari satu halaman ke halaman lain. Hampir semua website menggunakan link untuk navigasi.

Di HTML, link dibuat menggunakan tag <a>. Tag ini biasanya memiliki atribut href yang berisi alamat tujuan link.

Contoh Link


<a href="https://example.com">Kunjungi Website</a>

Pada contoh di atas, teks Kunjungi Website akan menjadi link yang bisa diklik. Ketika diklik, browser akan membuka alamat yang ada di dalam atribut href.

Link ke Halaman Lain

Link juga bisa digunakan untuk membuka halaman lain di dalam website yang sama. 


<a href="about.html">Halaman Tentang</a>

Jika link tersebut diklik, browser akan membuka file about.html.

Dengan menggunakan link, kamu bisa menghubungkan berbagai halaman sehingga website memiliki navigasi yang mudah digunakan.
