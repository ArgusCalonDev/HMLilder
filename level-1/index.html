Level 1

Pada level 1 kali ini, kamu akan belajar tentang dasar-dasar pemrograman web, mulai dari apa itu website, cara kerja website, dan lain sebagainya. Simak pembelajaran berikut agar kamu bisa melanjutkan ke level selanjutnya!

Daftar level 1

1. Pengenalan web

  • Apa itu website? Pengertian dan fungsinya
  • Bagaimana cara kerja website di internet?
  • Perbedaan website statis dan website dinamis
  • Apa itu domain dan hosting?
  • Apa itu browser?

2. Persiapan membuat website

  • Tools yang dibutuhkan untuk membuat website
  • Cara membuat website HTML pertama
  • Struktur dasar file HTML
  • Cara menjalankan website di browser

3. Dasar HTML

  • Apa itu HTML?
  • Mengenal tag HTML
  • Struktur dasar HTML (html, head, body)
  • Heading di HTML
  • Paragraf dan Line Break
  • Komentar di HTML

4. Konten dasar di HTML

  • Membuat link (a)
  • Menampilkan gambar (img)
  • Membuat list (ul, ol, li)
  • Membuat tabel (table)
  • Membuat form dasar (input, button)

5. Struktur halaman website

  • Mengenal elemen layout HTML (header, nav, main, footer)
  • Cara membuat navigasi website sederhana
  • Cara menghubungkan beberapa halaman HTML

6. Pengenalan CSS (dasar)

  • Apa itu CSS?
  • Cara menghubungkan CSS ke HTML
  • Cara mengubah warna dan font dengan CSS
  • Cara mengatur ukuran dan spasi
