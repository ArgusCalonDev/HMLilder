Membuat Tabel di HTML

Tabel digunakan untuk menampilkan data dalam bentuk baris dan kolom. Dengan tabel, informasi bisa disusun lebih rapi dan mudah dibaca.

Di HTML, tabel dibuat menggunakan beberapa tag seperti <table>, <tr>, <th>, dan <td>.

Contoh Tabel Sederhana


<table>
    <tr>
        <th>Nama</th>
        <th>Umur</th>
    </tr>
    <tr>
        <td>Adi</td>
        <td>17</td>
    </tr>
    <tr>
        <td>Budi</td>
        <td>18</td>
    </tr>
</table>

Penjelasan Tag

<table> digunakan untuk membuat tabel.

<tr> digunakan untuk membuat baris tabel.

<th> digunakan untuk judul kolom.

<td> digunakan untuk isi data tabel.

Dengan menggunakan tag-tag tersebut, kamu bisa menampilkan data dalam bentuk tabel yang terstruktur.
