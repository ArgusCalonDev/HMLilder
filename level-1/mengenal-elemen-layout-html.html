Mengenal Elemen Layout HTML

Saat membuat halaman website, kita biasanya membagi halaman menjadi beberapa bagian seperti header, navigasi, konten utama, dan footer. Pembagian ini disebut layout.

HTML menyediakan beberapa elemen khusus untuk membuat struktur layout yang lebih jelas, seperti header, nav, main, dan footer.

Contoh Struktur Layout


<header>
    <h1>Website Saya</h1>
</header>

<nav>
    <a href="#">Home</a>
    <a href="#">Artikel</a>
</nav>

<main>
    <p>Ini adalah konten utama halaman.</p>
</main>

<footer>
    <p>Copyright 2026</p>
</footer>

Penjelasan Elemen

<header> biasanya digunakan untuk bagian atas halaman, seperti judul website atau logo.

<nav> digunakan untuk menu navigasi, yaitu kumpulan link menuju halaman lain di website.

<main> berisi konten utama dari halaman website.

<footer> biasanya digunakan untuk bagian bawah halaman, seperti informasi copyright atau kontak.

Dengan menggunakan elemen-elemen ini, struktur halaman website menjadi lebih rapi dan lebih mudah dipahami.
