Level 2

Pada level 2 ini kamu akan melanjutkan pemrograman website dari level 1. Kamu berada di tingkat pemula tingkat lanjut. Guna memahaminya, simak pembelajaran berikut.

Level 2 - Pemula Lanjut

1. Pendalaman HTML

  • Perbedaan div dan span
  • Mengenal atribut HTML
  • Elemen semantik HTML (section, article, aside)
  • Struktur halaman artikel yang baik
  • Cara menyematkan video dan audio di HTML

2. CSS Layout Dasar

  • Apa itu CSS layout
  • Mengenal display di CSS
  • Menggunakan display: block dan inline
  • Mengenal flexbox
  • Membuat layout sederhana dengan Flexbox

3. Styling Website

  • Mengatur background dengan CSS
  • Menggunakan border dan border-radius
  • Mengatur posisi elemen (position)
  • Pseudo class (:hover, :active)
  • Membuat tombol dengan CSS

4. Responsive Web

  • Apa itu responsive web design
  • Mengenal viewport
  • Dasar media query
  • Membuat layout responsive sederhana

5. JavaScript dasar

  • Apa itu JavaScript
  • Cara menambahkan JavaScript ke HTML
  • Variabel di JavaScript
  • Fungsi dasar di JavaScript
  • Event website (click, dll)
  • Mengubah HTML dengan JavaScript (DOM Dasar)
