Cara Menyematkan Video dan Audio di HTML

HTML menyediakan elemen khusus untuk menampilkan media seperti video dan audio. Elemen yang digunakan adalah <video> dan <audio>.

Menyematkan video

Untuk menampilkan video di halaman web, gunakan elemen <video> dengan atribut src untuk menentukan file video. 


<video src="video.mp4" controls></video>

Atribut controls akan menampilkan tombol play, pause, volume, dan kontrol video lainnya.

Menyematkan audio

Untuk menampilkan audio, gunakan elemen <audio>. 


<audio src="audio.mp3" controls></audio>

Sama seperti video, atribut controls akan menampilkan kontrol pemutar audio.

Menggunakan elemen <source>

Kadang sebuah media memiliki beberapa format file. Kita bisa menggunakan elemen <source> agar browser memilih format yang didukung. 


<video controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
</video>

Jika browser tidak mendukung format pertama, browser akan mencoba format berikutnya.

Kesimpulan

  • <video> → untuk menampilkan video
  • <audio> → untuk memutar audio
  • <source> → menyediakan beberapa format media
