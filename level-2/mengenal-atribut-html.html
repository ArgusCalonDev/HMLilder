Mengenal atribut HTML

Atribut HTML adalah informasi tambahan yang ditambahkan ke dalam sebuah tag HTML. Atribut biasanya digunakan untuk memberi pengaturan, identitas, atau perilaku tertentu pada elemen.

Atribut selalu ditulis di dalam tag pembuka, dan biasanya memiliki pasangan nama="nilai".

Contoh:


<a href="https://example.com">Kunjungi website</a>

Pada contoh di atas, href adalah atribut yang memberi tahu link tersebut akan menuju ke alamat mana.

Contoh atribut yang sering dipakai

  • href → menentukan alamat link
  • src → menentukan sumber gambar atau media
  • class → memberi nama class untuk styling CSS
  • id → memberi identitas unik pada elemen

Contoh penggunaan beberapa atribut:


<img src="gambar.jpg" alt="gambar pemandangan">

<p class="teks-biru">Ini paragraf dengan class.</p>

<div id="container">Ini sebuah container.</div>

Kesimpulan

Atribut HTML digunakan untuk memberikan informasi tambahan pada elemen. Tanpa atribut, banyak elemen HTML tidak bisa bekerja dengan maksimal (misalnya link tanpa href atau gambar tanpa src).
